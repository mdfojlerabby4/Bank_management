# 🏦 MFR Bank - Complete Transaction Management System

![MFR Bank Screenshot](https://raw.githubusercontent.com/mdfojlerabby4/Bank_management/main/screenshot.png)

A comprehensive, client-side web application for managing bank accounts, transactions, loans, and request queues. Built with pure HTML, CSS, and JavaScript, it offers a complete banking simulation with persistent local storage.

🔗 **Live Demo:** [https://mdfojlerabby4.github.io/Bank_management/](https://mdfojlerabby4.github.io/Bank_management/)

---

## ✨ Key Features

### 🧑‍💼 Account Management
- **Create Account** – Add new customer accounts with initial deposit
- **View All Accounts** – Display complete list of accounts with balances
- **Search Account** – Find accounts by number or name
- **Check Balance** – View detailed account balance in a modal popup
- **Delete Account** – Secure deletion with balance verification

### 💸 Transaction System (Queue-Based)
- **Deposit / Withdraw / Transfer** – Requests are added to priority or regular queues
- **Transaction History** – View all past transactions with timestamps
- **Undo Last Transaction** – Reverse the most recent transaction

### 🏦 Loan Management
- **Apply for Loan** – Submit loan applications with amount, term, and interest rate
- **Make Loan Payment** – Process partial or full loan payments
- **Loan Status** – Track principal, interest, total payable, and progress
- **All Loans Overview** – Complete list of all loans with details

### ⚡ Queue Systems (Priority & Regular)
- **Priority Queue** – For VIP/High and Medium priority requests (sorted by priority)
- **Regular Queue** – For Normal priority requests (FIFO order)
- **Approve or Remove** – Each request can be individually approved or removed

### 📊 Dashboard
- Real-time statistics:
  - Total Accounts
  - Total Transactions
  - Pending Requests (Priority + Regular)
  - Total Balance Across All Accounts
- Quick action buttons for all major operations
