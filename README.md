# Bank Transaction Management System

A banking system using **Linked List, Stack & Queue**.

## Features
- ✅ Create/View/Delete Accounts (Linked List)
- ✅ Deposit/Withdraw/Transfer (Stack)
- ✅ Request Queue System (Queue)
- ✅ Undo Last Transaction

## Tech Stack
HTML | CSS | JavaScript | GitHub Pages

## Data Structures Used
- **Linked List:** Account management
- **Stack:** Transaction history
- **Queue:** Request processing
